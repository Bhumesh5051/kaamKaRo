kaamKaRo - MERN Stack Task Manager
​kaamKaRo is a full-stack productivity application designed to help users organize their daily workflow with a focus on real-time tracking and security.
​🚀 Key Features
​Secure Authentication: Full login and signup functionality powered by JSON Web Tokens (JWT).
​Real-time Dashboard: Dynamic tracking of task statuses including Pending, Completed, and Overdue.
​Advanced Task Management: Users can add tasks with specific titles, detailed descriptions, priority levels (High/Medium), and due dates.
​Organized UI: A clean interface for managing multiple tasks simultaneously.
​🛠️ Tech Stack
​Frontend: React.js
​Backend: Node.js & Express.js
​Database: MongoDB Atlas
​Security: JWT (JSON Web Tokens)

├── frontend/    # React frontend
└── .gitignore   # Ignored files (node_modules, .env)
